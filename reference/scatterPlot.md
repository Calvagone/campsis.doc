# Scatter plot (or X vs Y plot).

Scatter plot (or X vs Y plot).

## Usage

``` r
scatterPlot(x, output, colour = NULL, time = NULL)
```

## Arguments

- x:

  data frame

- output:

  the 2 variables to show, character vector

- colour:

  variable(s) to colour

- time:

  the time to look at those 2 variables, if NULL, min time is used
  (usually 0)

## Value

a ggplot object

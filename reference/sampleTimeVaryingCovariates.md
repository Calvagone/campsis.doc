# Sample time-varying covariates.

Sample time-varying covariates.

## Usage

``` r
sampleTimeVaryingCovariates(object, armID, needsDV)
```

## Arguments

- object:

  time-varying covariates, data.frame form

- armID:

  treatment arm ID

- needsDV:

  append extra column DV, logical value

## Value

a data.frame

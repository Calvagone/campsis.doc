# Create NOCB settings.

Create NOCB settings.

## Usage

``` r
NOCB(enable = NULL, variables = character(0))
```

## Arguments

- enable:

  enable/disable next-observation carried backward mode (NOCB), default
  value is TRUE for mrgsolve, FALSE for RxODE

- variables:

  variable names subject to NOCB behavior (see vignette for more info)

## Value

NOCB settings

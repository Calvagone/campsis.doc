# Generate IIV matrix for the given Campsis model.

Generate IIV matrix for the given Campsis model.

## Usage

``` r
generateIIV(model, n, offset = 0)
```

## Arguments

- model:

  Campsis model

- n:

  number of subjects

- offset:

  if specified, resulting ID will be ID + offset

## Value

IIV data frame with ID column

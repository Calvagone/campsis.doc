# Simulation settings class.

Simulation settings class.

## Slots

- `default`:

  default settings of the simulate method

- `hardware`:

  hardware settings object

- `solver`:

  solver settings object

- `nocb`:

  NOCB settings object

- `declare`:

  declare settings (mrgsolve only)

- `progress`:

  progress settings

- `replication`:

  replication settings

- `internal`:

  internal settings

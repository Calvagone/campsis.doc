# Preprocess 'replicates' argument.

Preprocess 'replicates' argument.

## Usage

``` r
preprocessReplicates(replicates, model)
```

## Arguments

- replicates:

  number of replicates

- model:

  Campsis model (class 'campsis_model' or 'replicated_campsis_model')

## Value

number of replicates to simulate

# NHANES database (demographics and body measure data combined, from 2017-2018).

NHANES database (demographics and body measure data combined, from
2017-2018).

## Usage

``` r
nhanes
```

## Format

data frame

- BS_ID:

  Original identifier

- SEX:

  Sex: 1 for males, 2 for females

- AGE:

  Age in years

- BW:

  Body weight in kg

- BMI:

  Body mass index

- HT:

  Height in cm
